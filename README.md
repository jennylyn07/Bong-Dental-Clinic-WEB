# Bong Dental Clinic

A static website for Bong Dental Clinic showcasing services, providers, blog content, and contact information.

## Features
- Responsive navigation with burger menu
- Homepage carousel and service highlights
- Services: Dental Implant, Orthodontics, Oral Surgery, Cosmetic Dentistry
- Blog section with multiple posts
- Contact details and form (front-end only)
- Appointment and success pages

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Icons: Font Awesome (CDN)

## Project Structure
- Root HTML pages: index, about, service, blog, providers, faqs, appointment, success
- Styles: page-specific CSS files (e.g., main.css, about.css)
- Scripts: page-specific JS in [Scripts/](cci:7://file:///c:/Users/Welcome/Documents/GitHub/Bong-Dental-Clinic-WEB/Scripts:0:0-0:0)
- Assets: images and fonts in dedicated folders (e.g., [main_img/](cci:7://file:///c:/Users/Welcome/Documents/GitHub/Bong-Dental-Clinic-WEB/main_img:0:0-0:0), [about_img/](cci:7://file:///c:/Users/Welcome/Documents/GitHub/Bong-Dental-Clinic-WEB/about_img:0:0-0:0), [Services_img/](cci:7://file:///c:/Users/Welcome/Documents/GitHub/Bong-Dental-Clinic-WEB/Services_img:0:0-0:0), [Fonts/](cci:7://file:///c:/Users/Welcome/Documents/GitHub/Bong-Dental-Clinic-WEB/Fonts:0:0-0:0))

## Getting Started
1. Clone the repo.
2. Open [index.html](cci:7://file:///c:/Users/Welcome/Documents/GitHub/Bong-Dental-Clinic-WEB/index.html:0:0-0:0) in your browser.
   - No build step required. No server required.

## Development
- Keep styles and scripts page-scoped (one CSS/JS per page where possible).
- Place images in the relevant `*_img` folder.
- Use Font Awesome via CDN for icons.

## Deployment
- Static hosting friendly (GitHub Pages, Netlify, Vercel).
- Ensure relative paths to assets remain intact.
- If using GitHub Pages, set the site to serve from the repository root or adjust links accordingly.

## Known Limitations
- Contact form is front-end only (no backend submission).
- No analytics or SEO meta tags beyond basics.
